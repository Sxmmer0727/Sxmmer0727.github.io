第3篇博客：java的基础知识补充

2022-02-28 562 words
学习

对java的基础知识进行了复习，给出一些自己容易遗忘的知识点

基础知识

  • 1.java标识符的长度无限制
  • 2.java采用Unicode字符集，因此标识符也可以用中文字符但是不推荐
  • 3.float类型为四个字节，但是表示的数值范围比8字节的long的范围更大：因为float采用一部分位表示数字，另一部分位表示几次幂，所以精度会低，但是范围更大。
  • 4.char类型中只能由一个字符或者转义字符：比如char ch = ‘\u0000’，char类型用作数值计算时位int类型
  • 5.强制类型转换为截断操作，所以可能会导致精度的损失
  • 6.int类型的除法没有小数，余数被直接舍弃
  • 7.取模运算的结果与被模数的正负保持一致
  • 8.&&为短路与，前面的表达式判断不成立后，不会执行判断后面的表达式
  • 9.最高效的乘法操作为位运算符操作2*8的最快为 2 << 33
  • 10.交换两个数值类型数据的方式除了定义临时变量，还可以采用加减和异或操作实现
  • 11.三元运算符(条件表达式)？(表达式1):(表达式2)，条件表达式为true则执行表达式1，否则执行表达式2
  • 12.else if 如果有多个表达式满足只会执行第一条else if
  • 13.case如果不加入break则会从上至下执行直到碰到break或语句结束
  • 14.case中的数据类型只能为6种：byte\short\int\char\枚举类型\String
  • 15.switch中的default顺序可以随意放置
  • 16.二维数组的length为其所包含的一维数组的个数，即外层元素的个数，初始值为内层元素的地址值
  • 17.二分法查找、冒泡排序、快速排序

快速排序的算法思想

选取一个pivlot值，采用双指针，低位指针的是数值高于pivlot并且高位指针的数值低于pivlot则两指针所指向的数据交换。如此一轮后递归调用，直至排序完成。
